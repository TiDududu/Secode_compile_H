Primary functionality: This contract is a simple math calculator that allows users to perform basic arithmetic operations.

1. add, This function takes two unsigned integers as input, adds them together, and returns the result.
2. mul, This function takes two unsigned integers as input, multiplies them together, and returns the result.
3. div, This function takes two unsigned integers as input, divides the first number by the second, and returns the result. It does not handle divide by zero errors.
4. sub, This function takes two unsigned integers as input, subtracts the second number from the first, and returns the result. It does not handle negative results.

Primary functionality: The contract allows for dispersing ether and ERC20 tokens to multiple recipients in a single transaction.

1. disperseEther, disperses ether to multiple recipients in a single transaction.
2. disperseToken, disperses ERC20 tokens to multiple recipients in a single transaction after approval from the sender.
3. disperseTokenSimple, a simplified version of disperseToken that directly transfers ERC20 tokens to multiple recipients without the need for prior approval.
